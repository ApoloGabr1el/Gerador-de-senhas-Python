#  Gerador de Senhas Aleatórias em Python

Um script simples, rápido e eficiente em Python para gerar senhas seguras e aleatórias.

---

##  Como Funciona?

O script combina **letras (maiúsculas e minúsculas)** e **números** para criar uma combinação aleatória de 12 caracteres. 

Ele utiliza duas bibliotecas padrão do Python:
* `string`: Para mapear de forma limpa os caracteres permitidos.
* `random`: Para sortear os caracteres de forma imprevisível.

---

##  Tecnologias Utilizadas

* **Python 3.14**
* Biblioteca `string` (Nativa)
* Biblioteca `random` (Nativa)

---

##  Como Executar o Projeto

Clone o repositório:
git clone git@github.com:ApoloGabr1el/Gerador-de-senhas-Python.git

entre na pasta:
cd Gerador-de-senhas-python

execute:
python GeradorSenha.py

Exemplo de saída:
3mNvjqfNMuZk
